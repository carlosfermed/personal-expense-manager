# Archive Completed Specs Plan

## 1. Document The Workflow

Update repository working rules and technical specification language so completed feature and change specs may be archived while pending and active work remains visible in the main feature and change directories.

## 2. Archive Completed Specs

Create archive directories under `specs/features/` and `specs/changes/`.

Move only completed feature and change spec directories into the matching archive directory.

Keep each moved directory unchanged.

## 3. Verify Organization

List active and archived spec directories.

Confirm each archived spec still has `Status: completed` and still contains `spec.md`, `plan.md`, and `tasks.md`.

Confirm there are no non-archived completed specs left outside archive except during the change itself.
